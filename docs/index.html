// what it is

A retro music player for your own files.

No ads. No algorithm. No cloud. Just your music — playing from a folder on your phone, your USB-C drive, or your iCloud Drive — through a player that looks like it was beamed in from 1999.

// what's inside

Eight things HarmonIQ does that everything else stopped doing.

  • 📁

    Drive-portable library

    Index lives on the drive itself. Plug into any iPhone, the library shows up.

  • 💾

    External drives

    Pick any folder visible in Files — including a USB-C SSD — and HarmonIQ indexes it.

  • 🔒

    Background playback

    Lock-screen + Dynamic Island controls. Live Activity for the current track.

  • 🎛️

    Real 10-band EQ

    AVAudioEngine + AVAudioUnitEQ. Sliders move the actual signal.

  • 🎨

    Winamp skins

    Drop a .wsz in. 9 bundled, more by tap.

  • ✨

    Smart Play

    14 rule-based modes plus 3 AI modes — Vibe Match, Storyteller, Sonic Contrast.

  • 🧠

    On-device AI

    Optional Apple Intelligence backend. No API key, no network egress.

  • 📈

    16 visualizer styles

    From the classic spectrum to neon glow, waterfall, Lissajous figures.

// skins

Nine bundled. Bring your own .wsz.

HarmonIQ reads classic Winamp 2.x .wsz archives natively — bitmaps, palette, transparency mask. Tap the visualizer to cycle the 16 styles; the active skin's VISCOLOR.TXT palette colors them.

// visualizers

Sixteen styles, one engine.

Each style runs on the same audio tap — RMS + peak from the EQ output, fed into a 64-sample oscilloscope buffer and 24-band spectrum. SwiftUI Canvas does the drawing at 30 fps.

  • SPECTRUMOSCILLOSCOPEPLASMAMIRROR
  • RADIAL PULSEPARTICLESFIRESTARFIELD
  • OSC · NEON GLOWOSC · HARMONIC LAYERSOSC · MIRROR WAVEOSC · FILLED WAVE
  • OSC · RADIAL WAVEOSC · WATERFALLOSC · LISSAJOUSOSC · BEAT FLASH

Tap the visualizer in the player to cycle. Visualizers.swift defines the lot.

// make it yours

The architecture is documented. The patches are welcome.

HarmonIQ is open source under MIT. Three places the project actively wants contributions:

  1. Drop a skin in

    Put a .wsz in HarmonIQ/Resources/Skins/, run xcodegen generate, build. The skin parser handles atlases, palette, transparency. Or use scripts/fetch-extra-skins.sh with a URL list.

  2. Add a Smart Play mode

    Extend SmartPlayMode + SmartPlayBuilder.buildQueue in SmartPlay.swift. Pure functions, easy to test, instantly visible in the Smart Play tab.

  3. Write a visualizer style

    Add a case to VisualizerStyle, a @MainActor drawing helper in Visualizers.swift, a preview thumbnail. The engine already exposes 24-band spectrum, oscilloscope buffer, beat flag, and rolling history.

PRs welcome. The architecture lives in CLAUDE.md.
